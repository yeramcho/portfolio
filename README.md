### 제로웨이스트 쇼핑몰&커뮤니티 (스프링 팀프로젝트)
+ 쇼핑몰 : 스토어에서 제로웨이스트 관련 상품을 구매
+ 커뮤니티 : 일상에서 실천 중인 제로웨이스트 생활 방식을 챌린지 사진 게시판에 인증
### 담당역할
- 팀장으로서 프로젝트를 리드
- ERD 설계 및 Oracle Cloud를 이용한 DB 관리
- 챌린지 사진 게시판 게시글 및 댓글 CRUD 개발
- 프로필 페이지 개발
- 관리자 챌린지 게시글 관리, 신고 관리, 주문 관리 페이지 개발
- 카카오페이 API 연동
- 클라우드 서버에 배포
